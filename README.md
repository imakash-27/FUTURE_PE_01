# FUTURE_PE_01 — AI Website Copy Generator for Local Businesses

**Internship Track:** Prompt Engineering (PE) — Future Interns
**Task:** Generate AI-powered website copy for a real local business
**Business:** Prime Relocations — premium packers & movers, Nellore, Andhra Pradesh
**Tools Used:** Claude (claude.ai)

## Overview

This repository documents the process of generating a complete, website-ready copy set — homepage, service pages, and CTA sections — for a local packers & movers business, using structured prompt engineering rather than single-shot prompting.

## Skills Demonstrated

- Prompt engineering for web content
- Conversion-focused copywriting
- Client requirement translation into brand voice
- AI content workflow design

## What's Inside

- `deliverable.md` — Full homepage, services, and CTA copy
- Documented prompt sequence (persona prompt → section prompts)
- Notes on why each prompt decision was made
- `screenshots/` — rendered output and generation process

## Process Summary

1. Set persona + brand constraints in a system prompt
2. Generated homepage copy (hero, trust blocks, closing CTA)
3. Generated 4 service page descriptions, tone-matched per service
4. Generated 4 CTA variants for different site placements
